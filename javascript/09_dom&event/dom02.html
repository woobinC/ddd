DOM을 활용한 element 선택 방법

  • getElementById(''): id로 선택 - 개별 element 선택
  • getElementsByTagName: 같은 tag elements를 배열로 반환
  • getElementsByClassName: 같은 class 이름을 가진 elements를 배열로 반환
  • getElementsByName: 같은 name을 가진 elements를 배열로 반환
  • querySelector(""): CSS 방식으로 선택한다. 해당되는 첫 번째 element 하나 반환
  • querySelectorAll: CSS 방식으로 선택한다. 해당되는 모든 elements를 배열로 반환
