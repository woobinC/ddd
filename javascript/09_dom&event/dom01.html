DOM & Events

웹페이지가 로딩되면 브라우저는 해당 페이지의 Document Object Model을 만든다.

DOM 은 객체들의 트리 구조를 형성한다.

JavaScript로 DOM을 다루는 방법

  • 페이지 안의 모든 HTML 요소를 바꿀 수 있다
  • HTML 요소(태그)의 속성을 바꿀 수 있다
  • CSS를 바꿀 수 있다
  • 현재 페이지에 있는 HTML 요소와 속성을 없앨 수 있다
  • 현재 페이지에 없는 HTML 요소와 속성을 만들 수 있다
  • 모든 HTML 요소가 event에 대해 반응하게 만들 수 있다
  • 페이지 안에서 새로운 HTML 이벤트를 만들 수 있다
