자바 스크립트의 자료형

5가지 데이터 타입

  • string
  • number
  • boolean
  • object
  • function

상세 설명

  • number(정수(integer), 소수형(float)): 10, 012, 0xa
  • string(문자형): "햄릿은 '사느냐 죽느냐 그것이 문제로다'라고 말했지요"
  • boolean(논리형): true, false
  • object(json): javascript object notation => {"name": "kim", "age": 27}

6가지 object 타입이 있다

  • Object
  • Date
  • Array
  • String
  • Number
  • Boolean

값을 가질 수 없는 데이터 타입 2가지

  • null => 0도 아니고 undefined도 아닌 상태. 즉 초기화 되었으나 값이 없는 상태
  • undefined: 선언되었으나 초기화되지 않은 상태

typeof : 데이터 타입을 확인하기 위해서 사용하는 연산자

Type Conversion(타입 변환)

  • Number => String: String(number), number.toString()
  • Boolean => String: String(boolean), boolean.toString()
  • Date => String: String(date), date.toString()
  • String => Number: Number(string), parseInt(string), parseFloat(string)
